# Translate Plugin
A language translator plugin for obsidian.

## Features
1. Translate selected text from the note itself.
2. Replace selected text with translated text.
3. Copy translation of selected text to clipboard.
4. A side window for additional translations.

## API's Used
Translations are done through 3rd party API's that are listed below:

| API                                                  | Description                               |
| ---------------------------------------------------- | ----------------------------------------- |
| [Projectsegfau](https://translate.projectsegfau.lt/) | It uses google translate for translation. |
| [My Memory](https://mymemory.translated.net/)        | It has 5000 chars/day limit.              |
| [Lingva](https://lingva.ml/)                         | The server is down most of the time.      |
>[!info]
>The API's may be down and won't work sometimes.
